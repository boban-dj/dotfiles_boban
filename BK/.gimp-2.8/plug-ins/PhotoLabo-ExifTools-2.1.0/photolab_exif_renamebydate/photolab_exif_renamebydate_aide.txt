Objet : 
Un traitement par lot pour renommer vos fichiers en les préfixant par la date 
de la prise de vue en fonction du tag EXIF correspondant.

Dépendances :
Les installations suivantes seront nécessaires tant que Gimp ne permettra pas de gérer 
les données EXIF depuis un script.
Pour les données EXIF, il faut installer JHead : 
http://www.sentex.net/~mwandel/jhead/ 

Menu :
Filtres / Photolabo / Outils EXIF / Renommer par date


Variables :

Répertoire = répertoire de travail où sont stockées les images à traiter.
Il n'y pas de copies, les noms des photos originales sont modifiés.

Extension = seuls les fichiers portants cette extension seront
traitées
