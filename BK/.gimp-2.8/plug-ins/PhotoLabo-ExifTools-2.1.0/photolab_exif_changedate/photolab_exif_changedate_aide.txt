Objet : 
Un traitement par lot pour changer la date de prise de vue
indiquée dans le tag EXIF de vos photos.

Dépendances :
Les installations suivantes seront nécessaires tant que Gimp ne permettra pas de gérer 
les données EXIF depuis un script.
Pour les données EXIF, il faut installer JHead : 
http://www.sentex.net/~mwandel/jhead/ 

Menu :
Filtres / Photolabo / Outils EXIF / Changer la date


Variables :

Répertoire = répertoire de travail où sont stockées les images à traiter.
Il n'y pas de copies, les données EXIF des photos originales sont modifiés.

Extension = seuls les fichiers portants cette extension seront
traitées

Signe = indiquer le sens du décalage horaire

Jours / Heures / Minutes /Secondes = indiquer le nombre de jours,
heures, minutes et secondes du décalage horaire.
