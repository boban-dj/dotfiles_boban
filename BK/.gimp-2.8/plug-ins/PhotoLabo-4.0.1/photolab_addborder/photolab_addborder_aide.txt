Objet : 
Ajoute un cadre simple de couleur pleine autour d'une image. En option,
ajoute un titre dessous dans le style polaroïd. Travaille en mode solo ou en 
traitement par lot.
En mode solo, un nouveau calque "cadre" est créé en-dessous du calque actif.
Vous pouvez aussi exporter le résultat auquel cas le nouveau calque "cadre"
sera effacé


Menu:
Filters / Photolabo / Ajouter un cadre
Filters / Photolabo / Travail par lot / Ajouter un cadre

Utilisation :
Directement depuis Gimp vous pouvez travailller avec des fichiers avec calques 
multiples, le calque actif est celui utilisé pour créer le cadre. La taille du
calque n'a pas d'importance, le cadre sera autour et l'image sera redimensionnée
pour laisser apparaître le cadre. Pour un traitement par lot, vous êtes sensé 
utiliser des fichiers jpeg ou png (sans calques donc). Les fichiers xcf et gif
seront aplatis avant l'application du cadre


Variables :

Répertoire source = uniquement pour un traitement par lot, répertoire de travail 
où sont stockées les images à traiter.

Extension = uniquement pour un traitement par lot, seuls les fichiers portants 
cette extension seront traités

Répertoire de destination = uniquement pour un traitement par lot, l'image
nouvellement créé sera enregistrée dans ce répertoire. Les répertoires Source et 
Destination peuvent être identiques, l'image d'origine ne sera pas effacée. Le
nom de la nouvelle image est suffixée avec le mot CADRE

Taille du cadre = détermine la taille du cadre en pixels

Couleur du cadre = une couleur pleine est utilisé pour remplir le cadre, aucune
ombre

Ajouter une fine bordure noire = en option, une bordure de 1 pixel et tracée 
autour de l'image

Couleur de la bordure fine et du titre = cette couleur est utilisée pour tracer
la bordure fine et le titre s'il y en a un

Ajouter un titre = en option, un titre sous le cadre

Taille du titre = le bas du cadre est augmenté de cette valeur. Vous n'avez pas
besoin de vous précoccuper de la taille de la police. Elle est automatiquement
calculée pour s'ajuster à l'intérieur de l'image (tracée comme une ligne simple,
texte centré)

Titre = le texte du titre. Doit être UTF-8 ou encodé en ASCII (sans accents)

Police du titre = la police utilisée pour rendre le titre

Exportation = uniquement pour un mode solo, exporte le résultat au lieu de
créer un nouveau calque de cadre dans le fichier original

Répertoire d"exportation = uniquement pour un mode solo, comme le Répertoire de 
destination

Variables dans le script :

titleGap = réglé à 0,05. Il est utilisée comme une valeur d'éloignement du texte
à l'intérieur du cadre inférieur. Il ne doit pas être trop élevé sinon la taille
de police sera trop réduite

tagDefault = set to _("BORDER"). Le suffixe ajoité derrière chaque fichier
nouvellement créé
