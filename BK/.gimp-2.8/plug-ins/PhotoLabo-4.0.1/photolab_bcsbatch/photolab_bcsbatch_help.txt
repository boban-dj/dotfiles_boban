Object: 
Apply Brightness Contrast and Saturation in batch mode

Menu:
Filters / Photolab / Batch corrections / Brightness Contrast Saturation


Parameters:

Directory = working directory in which the batch is running and the files to
handle are. The new created files are saved in the same directory and
prefixed with "BCS_", the old files are not deleted.

Extension = only the files with this extension will be treated

Luminosity = from -127 to +127. 0 value is neutral

Contrast = from -127 to +127. 0 value is neutral

Saturation = from -127 to +127. 0 value is neutral
