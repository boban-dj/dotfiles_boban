Objet : 
Traitement automatisé d'un lot d'images avec les outils de contraste, de
luminosité et de saturation de Gimp. 
« bcs » = Brightness Contrast and Saturation (pourquoi pas ?)

Menu :
Filtres / Photolabo / Corrections par lot / Luminosité contraste saturation


Variables :

Répertoire = répertoire de travail où sont stockées les images à traiter.
Les nouveaux fichiers sont placés dans le même répertoire et préfixés par
« BCS_ », les anciens fichiers ne sont pas effacés

Extension = seuls les fichiers portants cette extension seront
traitées

Luminosité = variation de luminosité de -127 à +127. La valeur 0 est neutre

Contraste = variation de contraste de -127 à +127. La valeur 0 est neutre

Saturation = variation de saturation de -100 à +100. La valeur 0 est neutre
