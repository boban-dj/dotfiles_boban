Object : 
Add a layer in "Softlight" mode
If a selection is present un layer mask within the selection is also created
and added

Menu:
Filters / Photolab / Softlight layer

Parameters :

Brightness = Luminosity given by the layer in softlight mode, 
from -127 to +127, 0 is neutral.
The layer is filled by a graytone color when dark=black, neutral=gray50%,
light=white


Notes :
Once the layer added you can always play with the fill color (in graytone)
to change the luminosity. Use preferably the Colorize tool of Gimp and
its Luminosity cursor.
