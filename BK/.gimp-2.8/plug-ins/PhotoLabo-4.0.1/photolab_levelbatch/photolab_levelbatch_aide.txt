Objet : 
Traitement automatisé d'un lot d'images avec un fichier enregistré depuis
l'outil Niveaux de Gimp. Fonctionne avec Gimp-2.6. Ne peut plus lire les
courbes de Gimp-2.4 et antérieur.

Menu :
Filters / Photolabo / Corrections par lot / Niveau


Variables :

Répertoire = répertoire de travail où sont stockées les images à traiter.
Les nouveaux fichiers sont placés dans le même répertoire et préfixés par
« LEVEL_ », les anciens fichiers ne sont pas effacés

Extension = seuls les fichiers portants cette extension seront
renumérotés

Fichier de Niveaux Gimp = le fichier enregistré depuis l'outil 
Niveaux de Gimp ( Outils / Outils de couleurs / Niveaux ). Les fichiers de 
Niveaux sont enregistrés par défaut dans votre répertoire utilisateur sous 
« .gimp-2.6/levels »


Problème :
Si le fichier Niveaux n'existe pas il ne se passe rien !
