Objet : 
Renomme une série de fichiers ayant tous la même extension EXT en fonction de 
l'ordre alphabétique. Le renommage s'effectue en utilisant un préfixe PREFIX
de la façon suivante : PREFIX_numero.EXT

Menu :
Filtres / Photolabo / Travail par lot / Renommer des fichiers


Variables :

Répertoire = répertoire de travail où sont stockés les fichiers à renuméroter
Les nouveaux fichiers sont placés dans le même répertoire, les anciens 
fichiers sont effacés

Extension = seuls les fichiers portants cette extension seront
renumérotés

Préfixe = le préfixe qui commencera les noms des nouveaux fichiers.


Alternatives :
Fichier «prefix.txt» = s'il est trouvé dans le répertoire de travail, son
contenu sera utilisé pour préfixer les nouveaux fichiers. Le paramètre
Préfixe n'est dès lors plus utilisé. L'encodage utilisé pour écrire ce fichier 
doit être le même que celui de votre système.


Problèmes :
Si l'encodage du fichier «prefix.txt» n'est pas celui du système,  
vous pouvez forcer manuellement l'encodage dans le Python-fu à la section 
« System locale »
